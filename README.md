# UC-JustJava

Aplikasi Android sederhana hasil project latihan.

## Paket
- `id.moklet.sigit.justjava`

## Deskripsi
Project ini adalah skeleton aplikasi Android dengan satu activity default.

Isi inti:
- `MainActivity` hanya menampilkan layout utama.
- Layout (`activity_main.xml`) menampilkan satu teks `Hello World!`.

## Struktur penting
- `app/src/main/java/id/moklet/sigit/justjava/MainActivity.java`
- `app/src/main/res/layout/activity_main.xml`
- `app/build.gradle`

## Cara jalan
1. Clone repo
2. Buka di Android Studio
3. Sync Gradle
4. Jalankan aplikasi

## Catatan
Ini cocok dijadikan baseline/template untuk mulai mengembangkan fitur lebih lanjut.
